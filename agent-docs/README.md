# Agent Docs

This folder contains the practical guidance that used to live directly in `AGENTS.md`.

- `workflow.md`: how to inspect, edit, test, and report changes
- `commands.md`: build, lint, test, and single-test commands
- `style.md`: JS/Python style and architecture guidance
- `python.md`: Python runtime, TOML, and profile notes
- `javascript.md`: JS runtime and Mineflayer notes
- `testing.md`: test strategy and recommended validation order
- `security.md`: secrets, destructive actions, and sandbox cautions
- `command-catalog.md`: command registry, docs generation, and sync flow

If repo-local instruction files exist (`.cursor/rules/` or `.github/copilot-instructions.md`), treat them as higher priority.
